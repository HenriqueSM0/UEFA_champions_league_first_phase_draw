This is a project that you can simulate UEFA Champions League Draw :

1. Write teams in ranking order by your willing, or consulting oficial UEFA teams ranking in file (teams.txt).

2. You need to put teams in format (Name,Country)

Example :

Real Madrid,Spain
Manchester City,England
Bayern Munchen,Germany
...

3. You can put more than 36 teams, but you will need to select these before simulate, and the pots will consider the order of the file.

4. Result it is shown by each team and your opponents, with (A) or (H) that represents :
Home 🏠, Away ✈️.

5. To draw, you need to run main function of Window_1 class.

5. Get funny simulating the biggest football club competition of the world and waiting for an amazing season !!
